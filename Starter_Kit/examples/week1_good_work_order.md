# Week 1 Example - Good Work Order

Create a weekly KPI workflow for the COO using `week1_customers.csv`, `week1_sales.csv`, `week1_support.csv`, and `week1_metric_definitions.md`.

Acceptance criteria:

- Paid active accounts equals 6.
- Net June revenue equals 2660.
- Open support tickets equals 3.
- The output names any assumptions.
- The script can be rerun from a clean folder.

Constraints:

- Do not define active customer using recent login.
- Do not report gross revenue when the metric asks for net revenue.
- Do not ignore blank `closed_date` values in support tickets.
